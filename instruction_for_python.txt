Instalation soft:
1. install python (https://www.python.org/)
2. install LibreCad or other program for dxf file reader (https://librecad.org/)
3. install Qgis (https://qgis.org/)

Program settings:
1. Add python python libraries. Open command window (Win+R, cmd), write and run:
	pip install ezdxf shapely numpy requests rasterio matplotlib scipy pyproj
	//If you get text "python.exe -m pip install --upgrade pip" copy that and run also
	//If you get text "Successfully installed..." close command window.
2. To get the API_KEY for download DEM. Open site https://opentopography.org, create account and save your API_KEY.
3. In Qgis add plugins:
	- OpenTopography DEM Downloader (for download DEM)
	- DEMto3D (for export DEM to stl format)
	- Basemaps (for import global maps)

Working process:
STAGE 1.GET DATA FOR A CARDBOARD MOCKUP
1. To get the area of interest. 
 - Open site https://geojson.io/ and drow region.
 - Save coordinate your area in KML format and name file map.kml to path with python script.
2. To get your area in horizontals in .dxf format.
 - Open file DEM_to_contours.py in text redactor(notepad) and configure them:
	API_KEY = 'cxxx6f70cxxxx69167efxxxx384exxxx'
	KML_FILENAME = 'map.kml'
	CONTOUR_INTERVAL = 5 # висота перетину рельєфу
	SURFACE_SMOOTHING = 2.0  # Сила згладжування рельєфу (sigma)
	USER_SCALE = 0.1       # 10 метрів = 1 мм у DXF (масштаб 1:10000)
	MARGIN = 5              # Відступ між деталями (мм)
	MAX_ROW_WIDTH = 5000     # Ширина листа картону (мм)
 - Run DEM_to_contours.py and you get file dem_to_contours.dxf.
3. To get nesting file for cutting at laser.
 - Open dem_to_contours.dxf, delete full map in buttom list and save that with name for_nesting.dxf 
 - Open file nesting.py in text redactor(notepad) and configure them:
	SHEET_WIDTH = 1000   # Ширина листа
	SHEET_HEIGHT = 895  # Висота листа
	PADDING = 2          # Відступ між деталями та від країв
	MAX_FONT_H = 13      # Максимальна висота шрифту
 - Run file nesting.py and you get 2 files:
	- nested_for_cut.dxf (that file to use in laser)
	- for_nesting_numeric.dxf (that file to use when creating a mockup)
4. To get a coordinate grid and map of the area
 - Getting map. Open Qgis -> Plagins -> Basemap Management -> Load Basemaps and load Google Satellite
 - Getting DEM in Qgis. Open file map.kml whit area interested and go to Raster -> OpenTopography -> OpenTopography DEM Downloader
	- recommended use DEM Copernicus SRTM 30
	- select ecstend not large 10x10 km
	*(use your API_KEY)
 - Getting image your`s area interest with coordinate grid in jpg format. Open Progect -> Create mockup. 
 - After adjusting the grid and scale Mockup -> save as image.
 - Add imafe in CAD program and scale with data in file dem_to_contours.dxf
 - Set up for printing on paper.
